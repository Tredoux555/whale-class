# Montree Brand & Style Guide

## Overview
Montree is a multi-tenant SaaS platform for Montessori school management, focusing on child progress tracking, portfolio management, and parent communication.

## Brand Colors

### Primary Palette
- **Dark Teal**: `#0D3330` (primary brand color)
- **Emerald**: `#4ADE80` (accent color)

### Usage
- Dark teal for backgrounds, branded overlays, caption boxes
- Emerald for accents, highlights, CTAs

## Typography

### Primary Font
- **Font Family**: Poppins
- **Weights**: Regular, Bold

### Video Caption Style
- **Font**: Poppins Bold
- **Text Color**: White (`#FFFFFF`)
- **Background**: Dark teal box (`#0D3330`)
- **Style**: Clean, readable, professional

## Brand Identity

### Core Messaging
- **Tagline/Hook**: "Your complete Montessori classroom — set up in 3 minutes. 🌿"
- **Value Proposition**: Fast setup, comprehensive classroom management, child-focused progress tracking
- **Tone**: Professional, educational, nature-focused, efficient

### Domain & Web
- **Primary Domain**: montree.xyz
- **Previous**: teacherpotato.xyz (migrated away)

## Social Media

### Handles
- **Instagram**: @montreexyz
- **TikTok**: @montreexyz
- **Facebook**: facebook.com/montreexyz

### Content Guidelines
1. Lead with benefit/speed ("set up in 3 minutes")
2. Include nature emoji (🌿) in captions
3. End with montree.xyz CTA
4. Use ~7 relevant hashtags
5. Focus topics:
   - Classroom management
   - Montessori education
   - Teacher tools
   - Child progress tracking
   - Parent communication

### Example Caption Structure
```
[Hook/Benefit] 🌿

[Brief description of feature/value]

montree.xyz

#Montessori #MontessoriTeacher #EarlyChildhood #TeacherLife #ClassroomManagement #ProgressTracking #MontessoriEducation
```

## Video Branding Elements

### Shorts Format (9:16)
- Branded top bar: logo + hook text
- Branded bottom bar: features + CTA
- Intro card (beginning)
- Outro card (end)
- Captions: Poppins Bold, white on dark teal box

### Square Format (1:1)
- Branded bottom bar: features + CTA
- Intro card (beginning)
- Outro card (end)
- Captions: Poppins Bold, white on dark teal box

### General Video Guidelines
- Keep webcam/face visible when present
- Use brand colors (dark teal, emerald) for all overlays
- Always include montree.xyz in CTA
- Maintain professional, educational tone
- Emphasize speed/efficiency of platform

## Key Features to Highlight

### Core Platform Features
- 3-minute classroom setup
- Child progress portfolios
- 5 Montessori curriculum areas (Practical Life, Language, Sensorial, Mathematics, Cultural)
- Photo/media timeline
- Parent portal with invite codes
- AI teacher advisor (Guru)
- Weekly planning tools
- Progress tracking and mastery indicators

### Target Audience
- Montessori teachers
- Montessori school administrators
- Parents of Montessori students

## Brand Personality
- **Efficient**: Fast setup, streamlined workflows
- **Natural**: Nature-focused (🌿), organic aesthetic
- **Professional**: Clean design, educational authority
- **Supportive**: Teacher-focused tools, AI advisor
- **Child-Centered**: Progress tracking, portfolio approach
