HeyGen Production

From script to upload — the exact workflow.

Use Builder → Script to Video. Avoid Video Agent (it billed 54 credits last time without producing a video). Same workflow per video. Set up takes ~5 minutes once you have the pattern.

Step 01

Open HeyGen and navigate to Builder

HeyGen.com → log in → top nav → Create video → Templates & Avatars → choose Script to Video (NOT Video Agent — that's the credit-burn trap).

Step 02

Lock the avatar (do this once, reuse for all 12)

Pick a stock avatar. Recommendations for Montree's voice:

  • "Daniel" or "Tyler" — warm professional male, 40s, neutral US accent. Continuity across all videos matters more than perfection.
  • Front-facing half-body shot, plain background (HeyGen will let you swap to dark green).
  • If you later train a personal Tredoux avatar (60 credits), the same script copy-paste workflow applies — just swap the avatar reference. Budget note: training the avatar mid-batch leaves you under-credit. Plan for next batch.
Step 03

Lock the voice (do this once)

HeyGen voice library → filter Male / English / Warm. Test 3-4 options with the V01 script. Once you've found "the Montree voice," save the voice ID and reuse it on all 12 videos. Continuity over perfection.

Step 04

For each video — paste the script

From this playbook: tap the green Copy button on the script's "Full HeyGen script" block → paste into HeyGen's script field. Don't manually retype — the line breaks matter for the avatar's pacing.

Step 05

Set the music & background

HeyGen has a stock music library. Use the production specs on each card to pick the right vibe — cinematic build (V01), soft acoustic (V02), motivational drive (V10), etc. Background: deep forest green (#0a1a0f) or a soft natural background. Match the Montree brand palette.

Step 06

Set aspect ratio: 9:16 vertical

All shorts are 9:16. Two exceptions where 16:9 is worth the extra 8 credits: V02 (What Montree Is), V07 (Tracy), V08 (Magic anthem), and V10 (Founding Agent) — these are the LinkedIn anchors and benefit from horizontal versions.

Step 07

Generate. Wait 2-4 minutes. Download.

HeyGen will render. Download as MP4. Drop into ~/Desktop/Master Brain/ACTIVE/whale/marketing/videos/launch-batch/ with the filename V01-million-dollar-app.mp4 etc. Consistent naming makes posting easier later.

Step 08

Quick review pass before you upload

Watch each video once. Check:

  • Avatar lip-sync is clean (HeyGen sometimes slips on long words — re-render if so).
  • Music isn't overpowering the voice (drop music to 20-30% if it is).
  • End frame holds long enough on "montree.xyz" for someone to read it (2 seconds minimum).
  • No "uh" or stutter from the avatar (rare with Script to Video but check).
Total HeyGen time investment. ~5 min setup + ~3 min per video × 12 = ~45 minutes of active work spread across whatever days you want. Render time runs in parallel.

Platform Strategy

Why each platform is here.

TikTok

The discovery engine

Where the algorithm is most generous to new accounts. Vertical-native. Captions short. Hashtags inline. Post at 7-10am OR 7-9pm local time. Comment back within the first hour after posting — TikTok measures that signal.

Instagram Reels

The Montessori parent stronghold

Where the Montessori parent audience already lives. Same captions as TikTok. Post 11am-1pm OR 7-9pm. Cross-post to Facebook Reels in the same publish dialog (saves time, doesn't hurt the algorithm if done at upload).

YouTube Shorts

The long-tail search engine

YouTube Shorts get indexed forever. Optimised titles matter more here than captions. People still find these videos six months later. Post 12-3pm OR 5-8pm. SEO bonus: include "Montessori" + a specific descriptor in the title.

All 12 videos · long-tail
Facebook Reels

The older-Montessori-mom audience

Skews older. Montessori moms in their 40s. Higher conversion to demo requests on Facebook than TikTok. Cross-post from Instagram (same upload action). Don't post separately — duplicate-content penalty.

Engagement protocol

Reply within an hour

The algorithm rewards conversations. For the first hour after each post, reply to every comment — even "🌱" with a "🙏". Use the rest days (Wed in week 1, Sun, etc.) to circle back to all comments from the prior days. This is what separates the videos that pop from the ones that disappear.
