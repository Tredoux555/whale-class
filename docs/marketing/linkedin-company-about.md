## About section (paste into LinkedIn "Overview")

```
Montree is a school management platform built specifically for Montessori schools. It tracks each child's progress through the Montessori curriculum, captures observations directly from classroom photos, and produces the records parents and principals expect — without forcing teachers to adapt to software designed for conventional schools. The platform is multi-tenant: a single school, a network of schools, or a homeschool can each run independently.

Montree is built by Tredoux Willemse, an AMS-certified Montessori Young Learner Specialist currently teaching a PreK 4 class in Beijing. The premise is simple: school management software for Montessori schools should be built by someone who teaches in one. What counts as a presentation, how a child's mastery is recorded, what a parent letter should sound like — these are pedagogical decisions, not engineering ones. Every feature in Montree comes from the gap between what a Montessori teacher actually needs in a classroom and what the existing software was built to do.

Montree serves Montessori schools, classroom teachers, and homeschool families looking for software that fits how Montessori actually works. The platform supports twelve languages and is operated by Montree Limited, a private company incorporated in the Hong Kong Special Administrative Region. To learn more or arrange a demo, visit montree.xyz or contact hello@montree.xyz.
```
