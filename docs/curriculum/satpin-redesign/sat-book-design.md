# WEEK 3 READER (REPLACES "Segina Sat!") — "The ___ Sat!"
*(Decided by Tredoux 2026-07-22: the first reader at t uses the sat frame with known oral vocab.
Segina Sat! is SHELVED — its generated art stays in the manifest for possible later use; Segina
remains in the alphabet cast scenes. This file follows the reader-designs-v2 conventions.)*

**Format:** hybrid (teacher narration + one child-owned decodable word per spread).
**Child's owned word:** `Sat` — s,a,t are the first three letters taught; this is the first
"I read that!" decode. **Frame:** teacher reads "The ant…", child completes by decoding the big
word **`Sat!`** — reads as one whole sentence. Nouns are picture-supported oral vocabulary the
child already knows from the s/a song weeks + the cast; the child names the noun from the art,
then decodes `Sat!`. **Layout:** teacher narration small grey type up top; `Sat!` BIG; noun never
printed (not decodable — the picture IS the noun).

class shout, no fake decode.
**Flashcard list:** `Sat` (NEW hero word), `at` (NEW), `a` (REVIEW) + oral picture cards: ant,
snake, apple, sun, star, cat, potato — all from THIS book's own page art (per the corrected
shelf rule: the shelf chain ends in the child reading this book).
