# Reverse-Engineering the Craft of SATPIN-Stage Decodable Readers

**Purpose:** Extract transferable design patterns from the world's best-regarded earliest decodable reader series (roughly the SATPIN stage: first ~6 letters s, a, t, p, i, n). Structures and parameters only — no reproduction of copyrighted text.

**Research date:** July 2026. Compiled from publisher pages, teacher guides, distributor listings, professional reviews, teacher blogs, and free open-licensed samples (notably SPELD SA's open PDFs, read page-by-page).

**A word on "SATPIN":** No two series use the identical 6-letter grouping. The common early sets:
- **Jolly Phonics / classic:** s, a, t, p, i, n (true SATPIN)
- **Sounds-Write (free SPELD SA PDFs):** a, i, m, s, t → n, o, p
- **Phonic Books Dandelion:** s, a, t, i, m → n, o, p → b, c, g, h
- **Little Learners Love Literacy:** m, s, f, a, p, t, c, i (holds **n** to Stage 2)
- **RWI Set 1:** m, a, s, d, t, i, n, p…
- **Collins Big Cat Pink A:** s, a, t, p, i, n, m, d
- **UFLI:** a, m, s, t, p first, then i, n

All name the same pedagogical window: the first ~8 single-letter sounds that yield VC/CVC words (sat, pin, nap, tin, mad).

# PART 1 — PER-SERIES NOTES

## Bob Books — Set 1: Beginning Readers (Bobby Lynn Maslen / Scholastic)
**The minimalist archetype.**

- **Set / length:** 12 books, ~12 pages each, roughly **7–20 words per book**. Small format (~11×14 cm). Book 1 (*Mat*) uses only 4 sounds: m, a, t, s — pure SATPIN core. Set adds all letters except Q across the 12 books, cumulatively; each book carries a prerequisite-sound index.
- **Page grammar (signature = combinatorial accretion):** single word → two-word sentence ("Mat sat.") → introduce second name → repeat two-word sentence ("Sam sat.") → **combine into a longer sentence** placing characters in relation ("Sam sat on Mat."). The difficulty curve is *horizontal* — more recombinations of the same tiny word set — not vertical (sentences stay 2–4 words; the count of known words in play accumulates). Across the set, books 1→12 grow from CVC toward two-clause ideas.
- **Story/humor engine:** tiny recurring cast (Mat, Sam, Dot the dog, Mac). **Physical pile-up gags** — one character sits on / bumps / stacks on another. The **picture delivers the punchline the flat text only sets up**; humor lives in the art-text gap.
- **Art:** sparse black-and-white line + a single spot color. Deliberately austere "to focus attention on decoding, not pictures." Because art shows a *consequence* rather than a literal label, the child usually **can't guess the words from the picture** — a deliberate anti-cueing (Science of Reading) stance.
- **Non-decodable support:** almost none at this stage — Set 1 is nearly pure CVC. Irregular HF words (the, on, was) used sparingly and added later. No adult read-aloud lines. **Titles are decodable by design** (Mat, Sam, Dot) — the title is the first decoding win.
- **Praise:** cheapest, most systematic; the "I read a whole book myself" confidence moment (recurring parent quote). Kids find the gags funny even when adults don't.
- **Complaints:** dull art ("bland," "leaves much to be desired"); text feels stilted to adults; kids outgrow Set 1 in ~2–3 months; best paired with letter tiles/flashcards.

## Half-Pint Kids — Level A "Sights & Sounds," Set 1 *Forest Fun* (LuAnn Santillo)
**The plot-forward miniature.**

- **Set / length:** whole program = 108 titles (Levels A/B/C). Level A = 36 titles in 6 thematic sets; Set 1 = *Forest Fun*. **7 pages per book** (shortest of any series studied), 1–2 short sentences per page, large text.
- **Sound set (Set 1):** b, t, h, c, p, m, f, s + short ă, ŏ. A child can read Book 1 **before mastering the whole alphabet** — an explicit early-confidence lever.
- **Page grammar:** title page → author line → **inside-front-cover "new words" list** → 7 illustrated pages (1–2 sentences each) → comprehension questions at back. Sentence density stays deliberately flat; the arc is a **problem→solution plot**, not escalating sentence length. New words front-loaded on inside cover so page text stays clean.
- **Story/humor:** recurring "Half-Pint" characters + per-set themes; characters **face the child** (designed eye-contact device). Publisher and independent reviewer both stress "real stories with a problem and solution" — unusually plot-forward for the word budget. Humor is character/situation-based, not wordplay.
- **Art:** colorful cartoon, "adorable and memorable" — opposite pole from Bob Books. Illustrations **directly reinforce** the decoded word (lessons have kids decode a CVC word and match it to the picture) — closer to picture-*confirmation* than anti-cueing.
- **Non-decodable support:** small explicitly-listed sight-word set surfaced inside the front cover (I, see, a, on, the, is, can, not, am); 33 sight words across Level A. Back-of-book comprehension questions double as an adult script. Theme name lives on the set; each book previews its own new words.
- **Praise:** "students love reading… fun, easy-to-decode"; "being able to read a whole book is very motivating"; adorable art; affordable enough to buy multiples for small groups.
- **Complaints:** very little independent critique exists (publisher review page is all praise) — treat glowing tone with caution; a genuine verification gap.

## Flyleaf Publishing — Emergent Reader Series (Laura Appleton-Smith)
**The "real picture book" decodable.**

- **Set / length:** 41 titles, sold as 6 book-sets (Set 1 = Books 1–5: *I Am Sam, Cam and Sam, Ann Can, C and K, Can You See 3?*). Per-book page count ~16 (unverified). Priciest of the field (~$4/book).
- **Sequence:** opens on s, m, ă (Book 1) and reaches /t/ only by Book 5 — a **slower, more spread-out** consonant rollout than Bob Books. Each book has a target sound + assigned HF "puzzle" words.
- **Page grammar:** back cover lists "High-Frequency Puzzle Words," new ones highlighted, so difficulty is signposted book-to-book. Text aims for **continuous narrative prose** rather than list-like lines — reviewers praise that it "avoided the halting, disjointed text" of rivals.
- **Story/humor:** character-and-situation comedy — recurring cats (Sam, Cam), dogs (Dot, Dan); gentle mysteries/mishaps ("a cat detective who investigates mysterious tacks"). Quieter/more narrative than Bob's slapstick.
- **Art:** the standout differentiator — multiple professional illustrators, "no stick figures or cartoons," "feel like real picture books." Art shows **MORE than** the sparse text (full scene, mood, subplot). Because text is fully decodable, lush art is "allowed" without inviting guessing.
- **Non-decodable support:** explicit "puzzle"/heart HF words per book, back-cover listed, **flagged NEW vs REVIEW**. Ships with teacher guides + Word Chain decks (adult scaffolding, not in-book read-aloud lines). Companion Books give younger siblings an abbreviated version of the same story.
- **Praise:** best-in-class illustration; genuinely story-like text; diverse representation "without fanfare."
- **Complaints:** thin coverage per phonics pattern; sequence "advances rather quickly" (not enough repetition for a struggler); most expensive.

## Phonic Books — Dandelion Launchers, Units 1–3
**The blending-rigor + dyslexia-friendly production model.**

- **Set / length:** Units 1–3 = 12 titles, **4 books per unit**, ~12 pages each (144 pp boxed). Design gives **two books per stage** — different stories over the *same* sound set (practice without new load).
- **Sounds by unit:** Unit 1 = s, a, t, i, m; Unit 2 = n, o, p; Unit 3 = b, c, g, h.
- **Page grammar:** hard constraint — **one line of text per page** at the Launcher tier (Dandelion *Readers*, the next tier, move to two+ lines). Titles reveal the arc: single-word label (*Sam, Bob, A Man*) → article+noun phrase (*A Map*) → short subject-predicate sentence / yes-no question (*Is It Sam?, On the Mat, Pam and the Cat*). Estimated ~8–15 unique words per book (unverified).
- **Story/humor:** small reused cast across all 12 (Sam, Tam, Tim, Pam, Pip, Bob + a bat, a cat). Humor is **thin by design** at this tier — the brand prioritizes blending practice and *avoiding predictable sentence frames* over gags; story/entertainment carried by the illustrations. Twist endings are a *higher*-tier feature, not verified at Launchers.
- **Art:** distinctive **color photo-montage/collage** (real photographic elements composited). Dyslexia-friendly production: **cream (not white) background**, clear accessible font. Explicitly anti-guessing ("blending sounds, not guessing"); art enriches/entertains, not a decoding crutch.
- **Non-decodable support:** "tricky words" introduced gradually. Earliest heart-word set from titles ≈ **I, the, a** (kept minimal). Reading game included; comprehension prompts/teaching notes live in a *separate* workbook. No evidence of in-book speech bubbles or adult read-aloud lines at this tier.
- **Praise:** systematic, genuinely decodable, "brilliant value"; dyslexia-friendly; avoids repetitive frames so kids must actually blend; works for reluctant KS2 and EAL.
- **Complaints:** thin in-pack teaching notes ("lacking meat on the bones"); comprehension questions sparse; photo-montage art is polarizing (some prefer drawn illustrations).

## Little Learners Love Literacy — Stage 1 ("Pip and Tim")
**The character-continuity + humor/twist model.**

- **Set / length:** Stage 1 = 5 books (*Sam, Pip, Tim* · *I am Tim* · *My cat Tip* · *The map* · *Sam is it!*). Full Pip and Tim = 61 books across Stages 1–7. "Little Book" stapled format; exact page count unpublished.
- **Sounds (Stage 1):** m, s, f, a, p, t, c, i (8 graphemes). Covers s,a,t,p,i of SATPIN; **n deferred to Stage 2** (adds b,h,n,o,d,g,l,v).
- **Page grammar:** familiar layout — **illustration on one page, text opposite**; short simple sentence(s) with high repetition of the focus graphemes. Books grow in length/complexity across stages. Stage 1 already teaches full stop, question mark, and exclamation mark — the arc deliberately moves statement → question → exclamation across the set (title *Sam is it!*).
- **Story/humor (the differentiator):** recurring **Zan family** — Tim, Pip, Mum, Dad + pets Sam and Tip (cat). Explicitly markets "real stories with relatable characters and lots of humour," stories "based on real life events" with "**a twist at the end**." Joke structure = short setup + end-of-book twist/punchline. Independent reviewer (Spelfabet): "cute, funny."
- **Art:** bright quirky drawn/cartoon illustrations (contrast Phonic Books' photo-montage). Strong anti-guessing stance: "decode the text — no guessing!" and "There are NO tricks in our Little Learner books."
- **Non-decodable support:** term is **"heart words"** (not "tricky words"). Stage 1 heart words = **the, my, I** (just 3). Every book carries "How to enjoy this book" adult notes + comprehension/vocabulary questions + a "Speed words" practice page. Titles are themselves decodable+heart-word strings (*My cat Tip, The map*).
- **Praise:** genuinely decodable, funny/character-driven, "perfectly sequenced"; endorsed by SPELD bodies and The Reading League (credibility signal).
- **Complaints:** price (~AUD 37–39 for 5 small books); Australian idiom/accent (a US/UK consideration). Little negative craft critique exists.

## SPELD SA Phonic Books — Set 1 (FREE open-licensed PDFs) — READ PAGE-BY-PAGE
**The single richest structural source (actual pages inspected).**

Note: two products share the name. The **free-PDF current set** aligns to **Sounds-Write** (a,i,m,s,t / n,o,p — *not* SATPIN order). The older physical "Set 1" followed Jolly Phonics (true s,a,t,p,i,n) but its pages were not accessible. Data below is the free set, from actual PDFs.

- **Set / length:** Set 1 = 10 books, each a **12-physical-page** PDF (free PDF + text-only + PowerPoint + HTML versions). Titles: 1 *Tim sat*, 2 *Nan*, 3 *Tan-Tan*, 4 *Is it a…?*, 5 *Kip has a nap*, 6 *Zac the dog*, 7 *The big box*, 8 *Red sand*, 9 *Sant the ant is ill*, 10 *A bug in a mug*.
- **Physical page grammar (every book):** p1 cover/title; **p2 = adult "Learning to read" note** (blending scripts + "Readers may need help with these words" list); **p3–p10 = story** (8 spreads; Book 1 uses 6); p11 = illustrator bio + "About this series"; p12 = full sound-sequence chart + copyright.
- **Word budgets (real, countable):** words per page range from **1 to ~11** across the set. Book 1's entire story ≈ **12 running words, ~6 unique** (Sit, Tim, sat, It, is, Sam).
- **New-vs-review — printed explicitly:** each teacher sheet lists **warm-up words** (decodable review), **"beyond sequence" words** (a few the child can't yet decode — the adult just says them), and **phonics focus** (the new letters). Verified Book 5: warm-up = is/the/Kip/Ken/pup/lap/let/red/rug/log; beyond-sequence = has/his/he/laptop; phonics focus = k, l, u, r.

**Real page-by-page arcs (word counts per story page):**
- **Book 1 "Tim sat"** (flattest, max sentence 3w): p4: 2w → p5: 2w → p6: 3w → p7: 1w → p8: 2w → p9: 2w. Opens on a 2-word spoken imperative, never exceeds 3 words, closes on a 2-word resolution. "One decodable action per page."
- **Book 3 "Tan-Tan"** (sentences now 7–8w, 2 sentences/page): p3: 5w cast-label line → p4: 10w → p5: 9w → p6: 5w → p7: 11w (longest) → p8: 3w (short beat) → p9: 7w (onomatopoeia "Pat, pat, pat!") → p10: 3w payoff ("Tan-Tan sits."). Arc = cast label → growing two-clause sentences → deliberate short beat → onomatopoeia climax → 3-word payoff.
- **Book 5 "Kip has a nap"** ("grow the tail" on a fixed frame): p3: 4w → p4: 6w → p5: 7w → p6: 6w → p7: 8w → p8: 9w → p9: 7w → p10: 9w. Frame "…has a nap on ___" stays fixed; the prepositional tail lengthens and the nap locations escalate to absurdity (lap → kit bag → rug → laptop → log).

So SPELD SA runs **two nested arcs**: within a book (label → full sentences → resolution) and across the set (Book 1 capped at 3w → Book 5 reaching 9w multi-clause).

- **Signature device (Books 2–3):** a single decodable **"action header" word printed on its own line above the sentence** (e.g. *Sit* / *Tap* / *In*), then the sentence using it — target word shown in isolation *then* in context on the same page.
- **Story/humor on ~6 unique words:** recurring named characters (Tan-Tan a dog, Kip a pup, kids Tom/Nan/Mim/Gab/Pip); refrains + triads ("Pat, pat, pat!"); escalation gag (silly nap spots); mini twist/resolution ending; question-reveal structure (*Is it a…?* poses a 3-option decodable question, next page answers).
- **Art:** full color, credited per-book illustrators (Dave Atze noticeably more cartoon-polished); cover art by Aboriginal artist Elizabeth Close; characters/settings reflect Aboriginal (APY Lands) communities with Anangu consultation. Art relationship varies: mostly **MORE than the text** (the visual gag carries humor the words only name); in *Is it a…?* the picture is the **reveal/consequence** confirming the correct decoded option — image confirms, never substitutes for reading.
- **Non-decodable support:** glue words (a, the, is, it, on, in, and, to) surfaced for the adult on p2, not taught as in-story sight words. Explicit adult blending scripts on p2. Speech marks/dialogue from Book 1. Decodable titles.
- **Praise:** widely ranked the #1 free decodable resource ("most comprehensive"); free multi-format + worksheets + fluency charts; indigenous themes.
- **Complaints:** utilitarian (short, thin plot, print-and-fold); free set uses Sounds-Write order which surprises teams expecting Jolly SATPIN.

## Oxford / UK Mainstream (Phase 2)

**Read Write Inc. Set 1 "Red Ditty" Books (Ruth Miskin / OUP):**
- Book = 16 pages, contains **3 short "ditties"** + comprehension. Reinforces RWI Set 1 single letters (m, a, s, d, t, i, n, p…). Explicitly sold as the **bridge between single-word reading and full sentences**; sound/word practice at the front before the ditty.
- Signature glue-word system: **"Fred Talk"** (sound out via Fred the Frog), **"green words"** (decodable), **"red words"** ("you can't Fred a red" — sight words). Humorous ditties + cartoon line art.
- Complaints: rigid/scripted; ditties feel contrived vs "real" stories.

**Floppy's Phonics: Sounds & Letters (Debbie Hepplewhite / OUP):** Level 1+ = 6 books, 12 pages each, covers all Phase 2 graphemes. Established cast (Biff, Chip, Kipper, Floppy). Signature: **richly detailed scenes with the target grapheme's items embedded in the picture** (a sound-hunt) — art used for phoneme-spotting/oral work, separate decodable text for reading. Splits the two roles rather than letting pictures cue words.

**Songbirds Phonics (Julia Donaldson / OUP):** the "real author, real story" answer to dull decodables — narrative quality + rhyme from a celebrated author at Phase 2–3.

**Collins Big Cat Phonics for Letters & Sounds (cleanest UK band data):**
- **Lilac = Phase 1:** wordless books (story through pictures; pre-decoding oral-language build).
- **Pink A = Phase 2 first half:** /s/ /a/ /t/ /p/ /i/ /n/ /m/ /d/ + repetitive CVC; "Chant and Chatter"; **no common exception words at all**; 16 fiction + 16 non-fiction.
- **Pink B = Phase 2/3 bridge:** adds g,o,c,k,e,u,r,h,b,f,l + ck,ff,ll,ss; 20 fiction + 20 non-fiction.
- Recurring cast Sam, Pip, Mog (cat). Fiction/non-fiction pairing at every band.

## Cult indie / TpT favorites (why teachers rave)

- **UFLI Foundations decodable passages (free, Univ. of Florida)** — the most-adopted free US SoR text. 128-lesson program; Alphabet Unit = lessons 1–34; each passage = a CVC **title + ~4–6 short complete sentences + illustration space, ~30–50 total words**, heavy **repetitive sentence frames**. **Jumps to sentences early** (controlled by only-taught phonemes). Glue = **"heart words."** Rave reasons: free, exhaustively cumulative sequence, one passage per lesson, printable.
- **Miss Rhonda's Readers (Montessori)** — Set One = 8 books × 8 pages. "Every story is true to life" (realism, not absurdist CVC filler); premium full-color art + physical quality. Etsy 5.0★/138 reviews. Praised for confidence-building for strugglers.
- **Simple Words Books (Jodi Reiner)** — pushes decodables to **chapter-book length** for older struggling/dyslexic readers; deliberately **limited/no illustrations**, font ≥12, uncluttered (so older readers aren't babied). Rave = dignity + "I could read that! Are there more?"
- **Reading Elephant (free)** — 1–3 words/page, one sound at a time with mastery gate; zero-overload for true novices.
- **Others teachers name at this stage:** Just Right Reader ("funny stories, diverse characters," QR sound intros); Dash Into Learning ("most charming illustrations"); Decode by Heart / Jax & Ren ("irresistible characters," "just enough story words"); Whole Phonics (UFLI-aligned, "adorable and funny"); Pathways to Reading (stories that "actually tell interesting stories," built-in comprehension); Reading for All Learners / "I See Sam" (very few words/page); Informed Literacy ("star words" for irregulars); Dog on a Log (OG, dyslexia-parent favorite). Caution: The Measured Mom's famous free set is being **retired** by its own author as pre-deep-phonics — popularity ≠ good sequencing.

# PART 2 — CROSS-SERIES SYNTHESIS
## The 12 Design Patterns in Every Great SATPIN-Stage Reader

Each pattern below carries concrete parameters extracted from the series above.

### 1. Tiny book, tiny word budget, big finish-line proximity
Best entry books run **7–16 pages** (Half-Pint 7, UFLI/Miss Rhonda 8, Bob/Dandelion/SPELD SA 12, RWI 16) with **~6–20 unique words** and **~12–50 total running words** (SPELD SA Book 1 ≈ 12 running / 6 unique; UFLI passages ~30–50). The engineering target is a book a beginner can *finish* in one sitting the day they know 4–6 sounds.

### 2. The label → phrase → sentence → punctuated-sentence arc (within AND across books)
Two nested arcs. *Within a book:* p1 single-word label/cast line → 2-word phrase (article+noun) → short subject-predicate sentence → question/exclamation → short resolution. *Across a set:* Book 1 caps sentences at ~3 words; by book 5 they reach ~9-word multi-clause (SPELD SA verified). Make the rung explicit — RWI literally sells its Ditty tier as "the bridge between single words and sentences." Weak sets skip rungs; loved sets pace them.

### 3. One line of text per page at the true entry tier
Phonic Books Dandelion Launchers = strictly **one line per page**; SPELD SA Book 1 = 1–3 words/page; Reading Elephant = 1–3 words/page. Illustration and text kept on facing/separate zones (LLLL: picture one side, text opposite). Add a second line only once the first tier is mastered (Dandelion Readers, SPELD SA later books).

### 4. Combinatorial accretion, not vertical growth
Difficulty rises by **recombining a fixed tiny word set into new arrangements** (Bob Books: "Mat sat" → "Sam sat" → "Sam sat on Mat"), and by **lengthening a fixed sentence frame's tail** ("__ has a nap on ___" with an escalating location). Horizontal expansion keeps every word decodable while manufacturing novelty.

### 5. A small recurring cast is the cheapest story engine
Every loved series reuses 3–6 named characters across the whole set: Bob's Mat/Sam/Dot; Collins' Sam/Pip/Mog; LLLL's Zan family (Tim/Pip/Mum/Dad + pets); SPELD SA's Tan-Tan/Kip; Flyleaf's cats/dogs. Familiarity lets a child "know" a character before decoding and lets a plot land without new vocabulary. Prefer decodable character names (Sam, Tim, Pip, Nan).

### 6. Manufacture humor in the art-text GAP, with 4 reusable joke structures
With <10 unique words, the joke rarely lives in the words. The recurring structures: (a) **physical pile-up / slapstick consequence** (Bob's "sit on"); (b) **escalating absurdity on a fixed frame** (SPELD SA's sillier and sillier nap spots); (c) **end-of-book twist/punchline** (LLLL "a twist at the end," exclamatory title payoff); (d) **question → reveal** (SPELD SA *Is it a…?* poses, next page answers). In all four the **final picture delivers the punchline the flat text only set up**.

### 7. Picture as reward/consequence/confirmation — never as a word-guessing cue
Universal anti-cueing stance ("no guessing / no tricks" — LLLL; "blend, don't guess" — Phonic Books). Three legitimate art-text relationships, all defeating picture-guessing: art shows a **consequence** of the decoded text (Bob), art shows **MORE than** the text — full scene the words only name (Flyleaf, SPELD SA), or art **confirms** a decoded choice after the fact (SPELD SA reveal). Floppy's Phonics splits roles: art for sound-spotting, separate text for decoding. Because the words are fully decodable, even lush art can't substitute for reading.

### 8. Two art strategies, both loved — pick a lane
The field splits and both poles get raves: **(a) austere** (Bob's B&W + spot color; Simple Words' near-imageless chapter format for older readers) to force decoding and signal seriousness; **(b) premium/charming** (Flyleaf's "real picture book" pro illustrators; Half-Pint's adorable cartoon; Miss Rhonda's full-color realism) to drive engagement and "real book" feeling. Dyslexia-friendly production is its own lever: **cream (not white) backgrounds, clear accessible fonts** (Phonic Books).

### 9. A tiny, fixed, visibly-flagged glue-word core (~3–9 words)
Keep non-decodable "heart"/"tricky" words minimal and named: LLLL Stage 1 = exactly **3** (the, my, I); Dandelion Launchers ≈ **3** (I, the, a); Half-Pint Set 1 ≈ 9 (I, see, a, on, the, is, can, not, am); Collins Pink A = **zero**. Give the system a brand name and a visible NEW-vs-REVIEW flag: RWI **green words / red words**, UFLI/LLLL **heart words**, Flyleaf **puzzle words (NEW vs REVIEW highlighted)**, Informed Literacy **star words**.

### 10. Print the scaffolding metadata on the book itself
The pro move is exposing the sequence data in/on the book: **inside-front-cover new-word list** (Half-Pint), **back-cover target sound + decodable list + HF words with word count** (Flyleaf, Junior Learning), **p2 adult "how to read this" blending script + "words the reader may need help with"** (SPELD SA), **warm-up / beyond-sequence / phonics-focus word lists** (SPELD SA). This makes new-vs-review explicit and gives the adult a script without cluttering the story pages.

### 11. Strictly cumulative, decodable-title, "review-only" tail
Nothing appears that hasn't been taught (UFLI, SPELD SA, Collins "week-by-week progression"). **Titles are themselves decodable** (or decodable + a known heart word): *Mat, Tim sat, My cat Tip, Pat Tap Sat* — the title is the first decoding win, not a barrier. Give a set a **revision-only tail**: later books recombine all prior sounds with no new load (Junior Learning Phase 2 books 8–12; Dandelion's two-books-per-stage design).

### 12. The metric that actually drives adoption: "I read it myself"
Across every review corpus, the rave reduces to **confidence and independent success**, not comprehensiveness: Bob ("her face lit up"), Half-Pint ("reading a whole book is very motivating"), Simple Words ("I could read that! Are there more?"), Miss Rhonda's ("confidence grow so much"). Design backward from the earliest possible genuine "I read a whole real book" moment — few sounds, short book, real (not nonsense) mini-story, a picture payoff. Early *success* beats coverage.
