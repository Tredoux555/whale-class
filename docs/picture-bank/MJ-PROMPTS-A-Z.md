# Montessori Object Picture Bank — Midjourney Prompt Pack

**156 prompts · 6 holdable objects per letter (A–Z).** Real objects a child names on sight and can hold — buy a mini or 3D-print it. These are for the **Montessori shelf / three-part cards** — separate from the googly-eye Dark Phonics circle set.

**Style (already baked into every prompt below):** single real object, pure white seamless background, soft even studio lighting, no props / people / text, 3:2. Just paste a line into Midjourney.

**Tip:** run one letter at a time, keep the cleanest of the 4, drop it in a folder named by the word (e.g. `sock/`).
