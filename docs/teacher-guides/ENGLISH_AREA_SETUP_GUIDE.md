## PHASE 1: PRE-READING (Age 2.5-3.5)


### 1.1 Sound Games (I-Spy)

**Purpose:** Develop phonemic awareness - the foundation of reading

**Materials Needed:**
- 3 small baskets or wooden boxes
- Miniature objects (15-20 per box)
- Each object represents a clear phonetic sound

**Sound Game Box 1 - Beginning Sounds:**
```
apple, ant, ball, bat, cat, cup, dog, duck, egg, fish, 
fox, goat, hat, hen, ink, jam, key, kite, lamp, leaf
```

**Sound Game Box 2 - Ending Sounds:**
```
bed (d), cup (p), hat (t), dog (g), bus (s), 
pen (n), jam (m), bell (l), mop (p), rug (g)
```

**Sound Game Box 3 - Middle Sounds:**
```
cat (a), bed (e), pig (i), pot (o), cup (u)
```

### 1.2 Sandpaper Letters

**Purpose:** Multi-sensory letter learning (visual + tactile + auditory)

**Materials:**
- 26 lowercase sandpaper letters
- Wooden boards (~15cm x 10cm)
- Consonants on PINK boards, vowels on BLUE boards
- Storage box with compartments

**IMPORTANT:** Lowercase ONLY. We do NOT teach uppercase first.

**Presentation Order (by frequency):**
```
Set 1: s, m, a, t, p, n
Set 2: c, o, i, d, g, r
Set 3: e, b, f, h, l, u
Set 4: j, k, v, w, x, y, z, q
```

**ESL Challenges for Chinese Students:**
- /l/ vs /r/ - practice both extensively
- /th/ sounds - not in Mandarin
- Short vowels - Chinese has tonal but not short/long distinction

---

### 1.3 Metal Insets

**Purpose:** Prepare the hand for writing

**Materials:**
- 10 metal insets (pink frames, blue shapes)
- Wooden inset stand
- Short colored pencils
- Pre-cut paper squares (14cm x 14cm)

**The 10 Shapes:** Circle, Square, Triangle, Rectangle, Oval, Ellipse, Trapezoid, Quatrefoil, Pentagon, Curvilinear Triangle

## PHASE 2: ENCODING (Age 3.5-4.5)


### 2.1 Large Moveable Alphabet (LMA)

**Purpose:** Children BUILD words before they can write them

**Materials:**
- Large Moveable Alphabet set
- Consonants: BLUE, Vowels: RED
- Letters ~5-7cm tall
- Storage box with 26 compartments
- Multiple copies of common letters (e, t, a, s, n, r, o, i)

**Key Principle:** Encoding (building words) comes BEFORE decoding (reading). Counter-intuitive but essential Montessori.

---

### 2.2 Object Boxes (Pink Series)

**Purpose:** Match physical objects to CVC word spellings

**Materials:**
- Small wooden box with dividers
- 9-12 miniature objects per box
- Corresponding word labels

**Pink Series Word Lists (Short Vowel CVC):**
```
Short A: cat, bat, hat, mat, rat, van, pan, can, bag, rag
Short E: bed, red, pen, hen, jet, net, wet, leg, peg, web
Short I: pig, wig, pin, tin, fin, bin, sit, hit, lip, rip
Short O: pot, cot, hot, dog, log, fog, box, fox, mop, top
Short U: cup, pup, bus, sun, run, fun, rug, bug, hug, tub
```

---

### 2.3 Picture-Word Matching Cards

**Purpose:** Visual association between image and written word

**Materials:**
- Picture cards with image only
- Word cards with text only
- Matching control cards (image + word)

**DIY Instructions:**
1. Print images on one set of cards
2. Print words on separate cards
3. Create control cards with both
4. Laminate all cards
5. Store in color-coded trays by vowel sound

---

## PHASE 3: DECODING (Age 4.5-6)


### 3.1 Pink Reading Cards

**Purpose:** First independent reading experience

**Materials:**
- Cards with CVC words ONLY
- No pictures - child must decode independently
- Organized by vowel sound

**Progression:**
1. Single word cards
2. Word + object matching
3. Short phrase cards
4. Simple sentence cards

---

### 3.2 Blue Series Introduction

**Purpose:** Words with consonant blends

**Blue Series Patterns:**
```
Initial blends: bl, br, cl, cr, dr, fl, fr, gl, gr, pl, pr, sc, sk, sl, sm, sn, sp, st, sw, tr
Final blends: -nd, -nk, -nt, -mp, -sk, -st, -ft, -lk, -lp, -lt
```

**Example Words:**
```
blob, brim, clap, crab, drip, flag, frog, glad, grin
plan, prop, scan, skip, slam, snap, spot, stem, swim, trap
```

---

### 3.3 Sentence Building

**Purpose:** Combine words into meaningful sentences

**Materials:**
- Word cards (pink and blue series words)
- Phrase cards
- Sentence strips
- Small objects for acting out

## MATERIAL CREATION GUIDE (DIY)


### Creating Pink Series Reading Cards

**Tools Needed:**
- Printer
- Card stock (300gsm)
- Laminator + pouches
- Paper cutter or scissors

**Steps:**
1. Use Whale Label Maker or create in Word/Pages
2. Font: Century Gothic or similar (clean, lowercase)
3. Font size: 48-72pt
4. One word per card
5. Print on card stock
6. Laminate for durability
7. Cut to size (~10cm x 5cm)

### Creating Picture-Word Matching Cards

**Steps:**
1. Find clear images online (no busy backgrounds)
2. Create three card types per word:
   - Picture only card
   - Word only card  
   - Control card (picture + word)
3. Size: ~8cm x 8cm
4. Laminate all cards
5. Color-code by vowel sound (pink trim = short a, etc.)

### Creating Object Boxes

**Steps:**
1. Get small wooden compartment box (6-9 sections)
2. Source miniature objects from Taobao
3. Print matching word labels
4. One object + one label per compartment
5. Include control card with all matches

## CLASSROOM FLOW

### Work Cycle Integration

**Morning Circle (Group):**
- Sound games introduction
- I-Spy with classroom objects

**Individual Work Time:**
- Sandpaper Letters (1:1 presentation first)
- Metal Insets (independent after demo)
- Moveable Alphabet (free exploration after introduction)

**Small Group:**
- Object box work
- Reading card practice

### Environment Setup Tips

1. **Rug placement** - Language work happens on rugs, not tables (except metal insets)
2. **Lighting** - Natural light on the language shelf if possible
3. **Quiet zone** - Language area should be in a quieter part of classroom
4. **Flow** - Clear path from shelf to work rug to return
