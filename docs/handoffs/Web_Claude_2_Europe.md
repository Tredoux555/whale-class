## WHAT TO SEARCH
For EACH country, search these sources:
- Google: "[Country/City] international Montessori school"
- Google: "[Country/City] AMI Montessori school"
- montessori-ami.org/find-a-school (filter by country)
- montessori-international.com (search by region)
- For UK: also check montessori.org.uk (Montessori Schools Association)
- For Germany: also check montessori-deutschland.de
- For Netherlands: also check montessori.nl
- For Scandinavia: check each country's national Montessori association

## WHAT TO COLLECT FOR EACH SCHOOL
Put the results in a simple table with these columns:
| SchoolName | Email | Country | City | Website | ContactPerson | Accreditation | AgeRange | Notes |

## PRIORITY ORDER
1. **Chain organizations first** — Guidepost Montessori (if European presence), Maria Montessori School (London, 3 sites), Spring Education/LePort, any chain with 3+ campuses
2. **AMI Training Centre affiliated schools** — London, Munich, Amsterdam, Rome, Bergamo all have AMI training centres with affiliated schools
3. **AMI/AMS accredited schools** — the gold standard
4. **Premium international schools** with Montessori programs (targeting expat families)
5. **Well-established standalone schools** — especially those 20+ years old

## RULES
- ONLY include schools where you can find an actual email address (info@, admissions@, admin@, or a named person's email)
- No generic contact forms only — we need actual email addresses
- If a chain has an HQ email AND campus emails, include BOTH
- Note if a school is AMI, AMS, or nationally accredited
- Netherlands has the most Montessori schools per capita in the world — be thorough there
- UK has hundreds — focus on the top-tier ones (AMI accredited, London/major cities, international)
- For Scandinavia: many municipal Montessori schools exist — focus on private/international ones
- Aim for 10-25 schools per country minimum, more for UK (30+), Germany (20+), Netherlands (20+)
