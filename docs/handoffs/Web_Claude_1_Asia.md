- Google: "[Country/City] international Montessori school"
- Google: "[Country/City] AMI Montessori school"

- montessori-international.com (search by region)
- international-schools-database.com (filter Montessori curriculum)
- For China specifically: also search in Chinese "蒙特梭利学校 [city]" and check Etonkids, MSB, Casa dei Bambini chains

## WHAT TO COLLECT FOR EACH SCHOOL
Put the results in a simple table with these columns:
| SchoolName | Email | Country | City | Website | ContactPerson | Accreditation | AgeRange | Notes |

## PRIORITY ORDER
1. **Chain HQs first** — Etonkids (60+ China), Brainy Bunch (if they have Asia presence), any chain with 5+ campuses
2. **AMI/AMS accredited schools** — these are the serious ones
3. **Premium international schools** with dedicated Montessori programs (ISB, BISS, etc.)
4. **Standalone Montessori schools** with websites that list email contacts

## RULES
- ONLY include schools where you can find an actual email address (info@, admissions@, admin@, or a named person's email)
- No generic contact forms — we need actual email addresses
- If a chain has an HQ email AND campus emails, include BOTH (HQ row + individual campus rows)
- Note if a school is AMI, AMS, or WASC accredited
- For China: many schools use WeChat-only contact — still include them but note "WeChat only" in Notes if no email found
- Aim for 15-30 schools per country minimum, more for large countries (China: 50+, Japan: 20+)
