# Beginning Sounds — Full Word Journey Map

## Philosophy

Beginning Sounds is NOT a phonics decoding exercise. It is **vocabulary building** — the child meets objects, learns their names, and associates the first sound. These same words and pictures then reappear throughout the entire phonics journey (Pink → Blue → Green), giving the child a familiar anchor each time they encounter a new spelling pattern.

**The natural learning sequence:**
1. **Beginning Sounds** — "This is a drum. Drum starts with 'ddd'." (Age 2.5-3.5)
2. **Pink Series** — Child decodes CVC words: c-a-t, d-o-g (Age 3-4)
3. **Blue Series** — Child decodes blends: dr-u-m, st-a-r (Age 4-5)
4. **Green Series** — Child decodes phonograms: sh-ar-k, tr-ee (Age 5-6)

**Key principle:** The child learns to SAY a word before they learn to READ it, and they learn to READ it before they understand WHY it's spelled that way. The rules come last — as confirmation of what the child already intuitively knows.

**Picture consistency:** The SAME picture of a drum appears in Beginning Sounds, then again in Blue Series (dr- blends), then possibly in Command Cards ("Tap the drum"). The picture is the child's companion — a visual anchor that grows with them.

## Dictionary Design Notes

### Pink Object Box Dictionary
- Contains all Pink words (Pink1 + Pink2) organized alphabetically
- Each page: word + picture (SAME picture as 3-Part Cards, Object Boxes, etc.)
- Beginning Sound words that appear in Pink are marked/highlighted
- Child uses this as a reference companion while working through Pink Series

### Blue Series Blend Dictionary
- Contains all Blue words (Blue1 + Blue2 + Blue3) organized by blend type OR alphabetically
- Each page: word + picture (SAME picture as other Blue materials)
- Beginning Sound words that appear in Blue are familiar friends — "I know this! That's a drum!"
- Could include a "Beginning Sound" badge on words the child already knows from vocabulary work

### Picture Consistency Rule
**CRITICAL:** When downloading/creating pictures for the Pixabay rewrite (Priority #1), ensure EVERY word that appears in multiple materials gets ONE definitive picture that is reused everywhere:
- Beginning Sound card
- Object Box label
- 3-Part Card
- Dictionary page
- Command Card illustration
- Story illustration

This means the image download script should produce a MASTER image set keyed by word, not by phase.
