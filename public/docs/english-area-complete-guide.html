üéØ How to Present I-Spy (Complete Script)
  1. Invite the child: "Would you like to play I-Spy with me?"
  2. Set up: Place 5-6 objects from ONE sound basket on a mat
  3. Name each object: Pick up each one: "This is a mouse. This is a mug. This is a moon..."
  4. Introduce the sound: "I spy with my little eye something that begins with /mmm/" (say the SOUND, not the letter name!)
  5. Wait: Let child find the object. If stuck, offer: "Mouse? Mmmm-ouse?"
  6. Celebrate: "Yes! Mouse begins with /mmm/!"
  7. Continue: Do 3-4 rounds, then invite child to be the spy
üí° Critical: Sound vs Letter Name

Say /mmm/ NOT "em". Say /sss/ NOT "ess". Say /buh/ NOT "bee". The pure sound is what matters!

üéØ How to Present Moveable Alphabet
  1. Readiness: Child knows most letter sounds
  2. First word: "Let's build 'cat'. What do you hear first? /c/..."
  3. Find letter: Child retrieves from box
  4. Continue: "What next? /a/..." until complete
  5. DO NOT ask to read it back ‚Äî that comes later!
