Dark Phonics - Sing-Along Cards

A tiny song-card book for every song - just the chorus + a big funny picture. The kid reads it by singing.

The idea: the child already knows the chorus from the song, so the same words + a picture = instant reading. Each page shows the line (big, for Canva) + a picture prompt (Midjourney - bright/funny, shared --sref 8888). Use Page 1's character as --cref for the rest. Print as a little card book, or flip on screen while the song plays.
