Dark Phonics - Sing-Along Cards

One card per sound: the catch phrase + one picture that nails it. Sing it, show it, done.

Each card is just the catch phrase (the Canva text) + one Midjourney prompt that draws the chorus literally - the big, clear, funny picture a kid instantly reads. Keep --sref 8888 on every card so the whole set shares one look; add a fixed --seed if you want them even tighter. Bold line = Canva text; box = Midjourney prompt.
