📖 How to Print as an A5 Booklet

Step 1: Save as PDF

  1. Select A5 Booklet from the Page Size dropdown above
  2. Click "🖨️ Print / Save PDF"
  3. In the print dialog, set Destination to "Save as PDF"
  4. Set Paper size to A5 (148 × 210 mm)
  5. Set Margins to None
  6. Check "Background graphics"
  7. Click Save — you now have an A5 PDF

Step 2: Print as Booklet in Adobe Acrobat

  1. Open the saved PDF in Adobe Acrobat (Reader or Pro)
  2. Go to File → Print
  3. Set paper size to A4
  4. Under "Page Sizing & Handling", click "Booklet"
  5. Booklet subset: "Both sides"
  6. Binding: "Left"
  7. Orientation: Landscape (Acrobat usually sets this automatically)
  8. If your printer supports duplex: check "Print on both sides" and flip on short edge
  9. If single-sided printer: print "Front side only" first, flip the stack, then print "Back side only"
  10. Click Print

Step 3: Fold & Staple

  1. Take the printed A4 sheets and stack them in order
  2. Fold the entire stack in half (A4 → A5)
  3. Staple twice along the spine (use a long-arm stapler if available)
  4. Trim edges if needed for a clean finish
💡 Simpler alternative: If you don't need a folded booklet, just select A5, save as PDF, and print normally on A5 paper — one page per sheet. Punch holes on the left and use a ring binder or ribbon to hold it together. Children love flipping through their own little dictionary!
💡 No Adobe Acrobat? Use Chrome's built-in print: select A4 paper, set "Pages per sheet" to 2, and print double-sided. The pages won't be in booklet order, but you can cut and staple them. Or use the free tool BookletCreator (bookletcreator.com) to reorder pages for booklet printing from any PDF.
