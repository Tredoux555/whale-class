Dark Phonics - Books

A cozy-dark-funny picture book for every song - 49 in all, tied to each song's catch phrase.

How to use: one book per song. Each page shows the Text (goes in Canva) and an Image prompt (paste into Midjourney). Every prompt already ends with the shared style + --sref 7777 so all books match. On Page 4 the character appears - approve that image, then add --cref [its URL] to every other page so the character stays the same. Build the art in Midjourney, lay out + add text in Canva, export PDF.
