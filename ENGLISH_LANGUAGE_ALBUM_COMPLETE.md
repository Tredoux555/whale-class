### Vowel Challenges (CRITICAL)

### ğŸ¯ Presentation: The Six Stages

**Stage 1: Single Object (Beginning Sounds)**

Hold one object (e.g., a cat figurine). Say:

```
"I spy with my little eye something in my hand 
that begins with /k/."
```

Make it **impossible to fail**. Child names the object.

**Stage 2: Two Contrasting Objects**

Place two objects with **distinctly different** beginning sounds (not similar sounds like /b/ and /d/). Child selects the correct one.

**Stage 3: Multiple Objects (3-6)**

Increase difficulty gradually. Add descriptive clues:

```
"I spy something you can wear that starts with /h/."
```

**Stage 4: Ending Sounds**

```
"I spy something that ENDS with /t/."
```

**Stage 5: Beginning AND Ending**

```
"I spy something that starts with /f/ and ends with /sh/."
```
(fish)

**Stage 6: All Sounds (Segmenting)**

```
"Can you tell me ALL the sounds in the word 'cat'?"
```

Child responds: "/k/ /a/ /t/."
