# 🖨️ 3D Printable Montessori Classroom

## Mission
Democratize Montessori education by making every classroom material 3D-printable. A parent or school needs only a printer, filament, time, and these designs to build a complete Montessori environment.

**Traditional classroom materials: $10,000–$30,000+**
**3D printer + filament: ~$200 + ongoing filament costs**

---

## Architecture: The Five Shelves

### 1. 📖 LANGUAGE ← STARTING HERE

#### 1.1 Sandpaper Letters (Trilingual: English / Afrikaans / Arabic)
- **Design**: Stencil frame (letter outline cut out) + raised letter insert (puzzle piece)
- **Triple use**: (1) Stencil for spray paint / art (2) Puzzle frame (3) Tactile tracing with raised insert
- **Dimensions**: ~15cm × 10cm tile (traditional Montessori size)
- **Color coding**: Blue = English, Green = Afrikaans, Gold = Arabic
- **Sets needed**:
  - English: 26 lowercase + 26 uppercase
  - Afrikaans: 26 lowercase + 26 uppercase (same glyphs, different phonics)
  - Arabic: 28 letters (isolated form first, connected forms later)
- **Post-processing**: Optional sandpaper/felt on letter surface for extra tactile feedback
- **Status**: 🔲 Not started

#### 1.2 Moveable Alphabet
- **Design**: Loose individual letters for word building on a mat
- **Size**: ~5cm tall per letter
- **Sets needed**:
  - Latin set (shared English/Afrikaans): consonants in blue, vowels in red (×2 sets minimum)
  - Arabic set: isolated forms, consonants/vowels color-coded
- **Storage**: Print a compartment box with labeled slots
- **Status**: 🔲 Not started

#### 1.3 Object Boxes (Phonetic Matching)
- **Design**: Small 3D-printed miniature objects (~3-4cm) for sound games
- **Pink Series objects**: 3-letter CVC words (cat, cup, pen, bag, dog, hat, map, sun, bed, pot...)
- **Blue Series objects**: 4+ letter words with blends
- **Green Series objects**: digraphs and complex phonics
- **Trilingual labels**: Each object gets 3 label cards (English / Afrikaans / Arabic)
- **Status**: 🔲 Not started

#### 1.4 Letter Tracing Boards
- **Design**: Grooved/channeled letters on larger boards for finger tracing
- **Size**: ~20cm × 15cm per board
- **Depth of groove**: 2-3mm (finger-width channel)
- **Status**: 🔲 Not started

#### 1.5 Grammar Symbols
- **Design**: Classic Montessori grammar symbols (large red circle = verb, black triangle = noun, etc.)
- **Pieces**: noun, article, adjective, verb, preposition, adverb, pronoun, conjunction, interjection
- **Size**: ~5cm each, flat bottomed for standing on sentence strips
- **Status**: 🔲 Not started

#### 1.6 Pink/Blue/Green Series Materials
- **Design**: Word cards + matching objects + picture cards
- **Print**: Objects are 3D printed, cards can be printed on paper
- **Status**: 🔲 Not started

## Work Schema

Every printable work follows this data structure:

```
WORK = {
  id: "lang-sandpaper-a-lower",
  name: "Sandpaper Letter: a (lowercase)",
  area: "language",
  sub_area: "sandpaper_letters",
  language: "english",            // english | afrikaans | arabic | universal
  age_range: [2.5, 5],
  
  // Print specs
  files: ["stencil_a.stl", "insert_a.stl"],
  print_time_minutes: 45,
  filament_grams: 12,
  filament_cost_usd: 0.24,
  printer_bed_min_cm: [15, 10],   // minimum bed size needed
  recommended_layer_height_mm: 0.2,
  recommended_infill_percent: 20,
  supports_needed: false,
  
  // Assembly
  assembly: "none",               // none | glue | sand | paint | fabric | mixed
  post_processing: "Optional: apply fine sandpaper or felt to letter surface",
  additional_materials: ["sandpaper (optional)", "spray paint (optional)"],
  
  // Pedagogy
  presentation_guide: "Three-period lesson: 1) 'This is ah' 2) 'Show me ah' 3) 'What is this?'",
  direct_aim: "Learn letter symbol and associated sound",
  indirect_aim: "Preparation for writing and reading",
  prerequisites: ["sound_games"],
  extensions: ["trilingual_variant", "puzzle_mode", "stencil_art"],
  
  // Difficulty
  print_difficulty: "easy",       // easy | medium | advanced
  montessori_level: "primary",    // toddler | primary | lower_elementary | upper_elementary
  
  // Status
  status: "not_started"           // not_started | designing | testing | complete
}
```

## Trilingual Extension (Tredoux Special)

### English ↔ Afrikaans Shared Alphabet
Same Latin letters, different phonics. One print set serves both languages.
Key phonetic differences to highlight:
- 'g' = /g/ in English, /χ/ (guttural) in Afrikaans
- 'r' = /ɹ/ in English, /r/ (rolled) in Afrikaans
- 'y' = /j/ sound in Afrikaans
- Afrikaans-specific digraphs: aa, ee, oo, uu, ie, oe, ei, ui, ou

### Arabic Alphabet Extension
- 28 letters, isolated form first
- Right-to-left orientation on tiles (flip the stencil frame)
- Later sets: initial, medial, final forms
- Diacritics (harakat) as separate overlay pieces
- Color: Gold tiles to distinguish from Latin sets
