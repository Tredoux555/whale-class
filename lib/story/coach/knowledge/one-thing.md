# The ONE Thing — Gary Keller

**The focusing question:** "What's the ONE thing I can do such that by doing it everything else becomes easier or unnecessary?" Ask it of the day, the week, the project, the life.

**The domino effect:** Success is sequential, not simultaneous. Line up the dominoes — do the most important thing first, and it knocks down the next. Don't try to topple them all at once.

**Extraordinary results come from a narrow focus.** "You need to be doing fewer things for more effect instead of doing more things with side effects."

**Time-block the ONE thing.** Put it on the calendar first, early, and protect it like a meeting with yourself. Everything else fills in around it.

**Multitasking is a lie.** You're just switching fast and paying a tax each time. Singular focus on the lead-domino task is where leverage lives.

**The success habit:** ask the focusing question, find the lead domino, do it first, protect the block. Repeat daily.
